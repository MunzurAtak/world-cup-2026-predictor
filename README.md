# World Cup 2026 Match Outcome Predictor

This project predicts match outcomes for the FIFA World Cup 2026 using historical international football results.

## Goal

The goal is to build a data science system that can:

- Predict World Cup match outcomes
- Simulate the 2026 group stage
- Create predicted group rankings
- Build a knockout tournament bracket
- Display results in a Streamlit dashboard

## Tech Stack

- Python
- pandas
- scikit-learn
- Streamlit
- Plotly
- GitHub

## Project Structure

```text
app/                 Streamlit app
data/raw/            Original datasets
data/processed/      Cleaned datasets
notebooks/           Exploratory analysis
src/data/            Data loading and cleaning code
src/features/        Feature engineering code
src/models/          Model training and prediction code
src/simulation/      Group and knockout simulation logic
reports/figures/     Visual outputs
```
